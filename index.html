LiteGallery Demo

How to Use

        // Initialize gallery
        const gallery = new LiteGallery('.lite-gallery', {
            thumbnails: true,
            fullscreen: true,
            zoom: true,
            rotate: true,
            zoomLevels: [1, 1.5, 2, 3] // Custom zoom levels
        });

Note: Zoom by clicking is only available in fullscreen mode. Click on an image to enter fullscreen, then click again to zoom.

Library Features

  • Image Navigation: Navigate through images using arrows or swipe gestures
  • Fullscreen Mode: Click on image to enter fullscreen view
  • Zoom Functionality: In fullscreen mode, click to cycle through zoom levels, or use the mouse wheel
  • Image Rotation: Rotate images 90Â° at a time with the rotate button
  • Responsive Design: Works on both desktop and mobile devices
